# 要件定義書 (Requirements Definition)

## 1. アプリケーション概要
**名称**: Photo-Arranger
**目的**: 同一被写体で撮影時期の異なる2枚の写真の色調と明るさを合わせる。

## 2. 機能要件

### 2.1 画像読み込み
- **形式**: JPEG
- **枚数**: 2枚 (Target / Reference)
- **表示**: 左右に並べて表示

### 2.2 色調・輝度補正機能
- **自動補正 (Auto Match)**:
    - **カラーチャートあり**: 写真内のカラーチャート（例: Macbeth Chart）を検出し、色調と輝度を基準（または相互）に合わせる。
    - **カラーチャートなし**: 画像全体の色情報を分析し、色調を合わせる。
- **手動補正 (Manual Adjustment)**:
    - 各写真に対して個別に調整可能。
    - **調整項目**:
        - 色調 (Color Tone / Temperature / Tint)
        - 明るさ (Brightness / Exposure)

### 2.3 保存機能
- 各写真を個別に保存可能。
- 保存形式: JPEG

## 3. ユーザーインターフェース (UI)
- **メインエリア**: 2枚の写真（左：写真A、右：写真B）
- **コントロールエリア (中央/下部)**:
    - [Auto Match] ボタン
- **調整エリア (各写真の下)**:
    - スライダー: 色調
    - スライダー: 明るさ
- **フッターエリア**:
    - [Save Photo A] ボタン
    - [Save Photo B] ボタン

## 4. 非機能要件
- **動作環境**: Windows Desktop Application (Standalone)
- **レスポンシブ**: ウィンドウサイズに合わせてリサイズされること。
